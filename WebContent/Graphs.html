About Project

Primary objective of this project is to understand the Amazon AWS (EC2) cloud infrastructure and learn to configure the Infrastructure, Software and application layer for it. Secondary goal involves learning Hadoop and build meaningful application for demonstration purpose and deploy the same on Ubuntu 13 OS which is on Amazon Elastic Cloud 2. Also adding extra features like Hadoop load balancing and security if time permits.
